# Better Firefly
A Chrome extension that includes auto-login, dark mode and other helpful features for the Gordano Firefly Website.
Features include:
-Colour customisation
-Auto Login
-Dark node
-Custom Name
-Profile Picture remover
-Unnecessary dashboard clutter
